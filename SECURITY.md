# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 0.1.x | Yes |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by opening
a GitHub Issue with the `security` label. We will acknowledge receipt
within 48 hours and provide a fix or workaround within 7 days.

Do not disclose vulnerabilities publicly until a fix is available.
